GivLocal is free to use for monitoring. Two optional one-time purchases unlock additional control and configuration features.

## GivLocal Pro

Pro unlocks all control features:

- Charge and discharge schedules (create and edit slots)
- Quick actions — Charge Now, Discharge Now, Pause
- Battery reserve and power limit adjustments
- EV Charger controls
- Tailscale remote access
- Port forwarding remote access
- Set a favourite device for auto-connect on launch
- Rename devices and assign custom remote addresses

Pro is a **one-time purchase** — no subscription.

To restore a previous purchase, tap **Restore Purchase** on the Pro screen.

## GivLocal Installer

The Installer tier adds the **Commissioning** section to the Inverter tab, covering settings that are typically configured once during installation:

- Grid Code (grid compliance standard)
- Meter Type (CT clamps or energy meter)
- Battery Type and Capacity
- Auto Read BMS
- PV Startup Voltage
- Grid Export Limit
- Grid Import Current Limit
- PV Input Mode (independent strings or combined)
- CT Direction
- Force Off Grid

These settings are intentionally separated from the standard controls to reduce the risk of accidental changes.

The Installer tier is a **one-time purchase**. If you have already purchased Pro, a discounted upgrade price is offered when unlocking Installer.

To restore a previous purchase, tap **Restore Purchase** on the Installer screen.

<Aside type="tip">
  Commissioning writes show a confirmation prompt by default. This can be disabled in **Settings → Diagnostics → Confirm Commissioning Writes** for faster workflows during an installation session.
</Aside>
